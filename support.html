Getting Help#

For questions about how to use the Hub, please choose from the following resources

  • Always check the FAQ page first

  • Use the LEAP-Pangeo Discussion Forum to search for previous topics or post new ones

  • Join the LEAP Slack and post an issue in the '#leap-pangeo' channel.

  • Reach out to the Data and Computation Team by mentioning '@data-and-compute' on the LEAP slack channel

  • Tag '@leap-stc/data-and-compute' in a Github issue.
